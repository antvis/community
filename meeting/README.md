# AntV 项目社区周会记录

## 说明

AntV 项目社区周会是团队定期举行的会议，用于同步项目进展、讨论技术方案、分享经验和解决问题。

## 会议模板

```markdown
- **日期**: YYYY-MM-DD
- **主持人**: 姓名

## 议题

2. 技术方案讨论
3. 问题反馈与解决
4. 下周工作计划

### 项目进展
```
